# Настройка pgAdmin

## Автоматическая конфигурация

pgAdmin уже настроен с предустановленным сервером:
- **Имя сервера**: Postgres Air
- **Хост**: db (внутри Docker сети)
- **Порт**: 5432
- **База данных**: postgres_air
- **Пользователь**: postgres
- **Пароль**: postgres

## Вход в pgAdmin

1. Откройте http://localhost:8000
2. Войдите с учетными данными:
   - **Email**: admin@example.com
   - **Пароль**: admin

3. **Сохраните пароль в браузере** - после этого браузер будет автоматически вводить пароль при следующем входе.

## Мастер-пароль

Мастер-пароль отключен (`PGADMIN_CONFIG_MASTER_PASSWORD_REQUIRED: False`), поэтому при первом входе он не будет запрашиваться.

## Автоматический вход

К сожалению, полностью отключить веб-аутентификацию в pgAdmin 4 невозможно. Но вы можете:
- Сохранить пароль в браузере (автозаполнение)
- Использовать расширение браузера для автоматического входа
- Или просто один раз войти - сервер уже настроен и готов к работе
